# 主线状态

盖茨已确认重生，但没有向任何人透露。

未来记忆被确认为方向性记忆，不是可直接复制的技术答案。

微软当前仍是依赖MS-DOS、IBM PC、授权、移植、交付、维护的小公司。

DOS被确认为现金流、行业接口和未来布局地基，不能被贸然推翻。

IBM被确认为强势但非敌对的大客户，它正在逼迫微软建立测试、文档、责任边界和交付纪律。

软件复杂度被确认为长期核心敌人，但目前只通过支持记录分类来积累证据。

苹果线已正式登场。

Lisa证明图形化方向有价值，但路线过重。

Macintosh证明更低成本、更个人化的GUI路线存在可能，但仍处于早期原型阶段。

第010章完成第一次战略会议，但仅提出问题、记录方法和研究方向，没有批准Phoenix立项。
